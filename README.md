## Getting Started

1. 프로젝트를 클론
```bash
git clone https://github.com/jiyoon0605/jy-dev-blog.git
```
2. 필요한 환경 변수 설정
3. 의존성 설치 및 실행
```bash
npm install
npm run dev
# or
yarn
yarn dev
# or
pnpm install
pnpm dev
# or
bun install
bun dev
```
